AWARE Dashboard

Main page

This page gives you 3 options to access the main AWARE Dashboard endpoints: configurator, studies, and the setup wizard.

Configurator

Build and manage Android study configurations through the public configurator UI.

Open /configurator/

Studies

Browse the public JSON study files served from /studies/.

Open /studies/

Setup Configurator

Open the setup wizard container. This is available while the setup image is running on port 9999.

Open setup wizard

If the setup wizard is not running, the internal setup link will not be reachable until you start it again with ./setup.sh or setup.bat.
